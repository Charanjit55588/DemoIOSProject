# DemoIOSProject

Description: Automation project for mobile app. 

Purpose: Build a framework so we can use it anytime to automate a mobile app.

TechStack: Java, Maven, Appium, Xcode, Android Studio

Pre-requisite:
1. Install Java 
2. Set path for Java and Android studio in your profile or you may set up in the appium configuration

Steps to contribute:

1. Clone the repository.
2. Create a branch and start editing.
3. When you finish with the changes push the code to the remote and create a pull request for review.
4. Once the code is merged. You may delete the branch.
